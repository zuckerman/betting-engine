## Daily Routine

**Every morning:**

1. Check `/dashboard/validation` → **note the 4 metrics above**
2. Check `/dashboard/models` → **any new promotions?**
3. Check `/dashboard/evolution` → **what's the system health?**

**Only interfere if:**

- CLV turns sharply negative (< -0.02)
- Drawdown spikes (> 30%)
- System error (API down, etc)

**Do NOT change anything if:**

- Early data (< 100 bets)
- One bad day (variance is normal)
- Temptation to "optimize"

## The 14-Day Phases

### Phase 1 (Days 1–5): Data Collection
- Let system run
- Collect signals
- Don't draw conclusions yet

### Phase 2 (Days 5–10): Early Signal
- Pattern emerges
- Still too early to act
- Continue collecting

### Phase 3 (Days 10–14): First Real Conclusions
- Sample size approaching credibility
- Can start thinking about next steps
- Still run to Day 30+ if possible

---

## What You're Looking For

### ✅ Signs of Real Edge

- CLV positive and stable
- % beating market > 55%
- Drawdown predictable
- Models improving over time

### ⚠️ Yellow Flags

- CLV ≈ 0 (breakeven)
- Drawdown chaotic
- One model dominating (overfitting risk)

### ❌ Red Flags (Stop & Reassess)

- CLV negative (market beats you)
- Drawdown > 30%
- System promoting strategies that fail immediately
